# 🏛️ Rashtrapati Nilayam Website

A simple and interactive HTML project about **Rashtrapati Nilayam (Hyderabad)** created for a presentation.

---

## 📌 About the Project
This website provides information about Rashtrapati Nilayam, the official winter residence of the President of India.  
It includes structured sections, navigation, and interactive features.

---

## ✨ Features
- 📑 Multiple sections (Overview, History, Architecture, Gardens, Importance, Facts)
- 🧭 Smooth scrolling navigation bar
- 🎨 Styled layout with CSS
- 🔐 Password-protected secret section
- 👁️ Show/Hide password option
- ⏱️ Auto-hide secret content after a few seconds

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

---

## 🔒 Secret Feature
The website includes a hidden section that can be accessed using a password.

- Enter password to reveal hidden content
- Content automatically hides after 8 seconds
- Includes show/hide password toggle

---

## 🚀 How to Run
1. Download or copy the HTML file  
2. Open it in any web browser  
3. Use the navigation bar to explore sections
